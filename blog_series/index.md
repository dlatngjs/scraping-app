# 블로그 시리즈: 데이터 수집부터 시각화까지 (3부작)

이 시리즈는 **웹 데이터 수집 → 데이터 정리 → 시각화 자동화**까지 한 흐름으로 다룹니다.

## 1편. 웹 스크래핑 입문: 필요한 정보 빠르게 모으기
- 목표: 웹 페이지에서 제목/헤더/링크를 안정적으로 수집
- 다룰 내용:
  - Python 표준 라이브러리(`urllib`, `html.parser`)로 스크래핑
  - 간단한 CLI 도구 구조 만들기
  - 예외 처리(네트워크 오류, 인코딩 이슈)

## 2편. CSV 데이터 가공: 분석 가능한 형태로 정리하기
- 목표: 수집 데이터를 CSV 포맷으로 정리하고 품질 점검
- 다룰 내용:
  - 컬럼 설계(date, value 등)
  - 결측치/중복치 처리 아이디어
  - 재사용 가능한 데이터 로딩 함수 작성

## 3편. 추이 그래프 자동화: 보고서용 이미지 한 번에 만들기
- 목표: CSV를 입력받아 추이 그래프를 자동 생성
- 다룰 내용:
  - `matplotlib` 라인 차트 기본
  - 축/제목/격자/레이아웃 설정
  - PNG 파일 저장 및 자동화 실행
